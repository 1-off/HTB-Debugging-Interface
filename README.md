# HTB-Debugging-Interface

- download Logic2
- read [uart-a-hardware-communication-protocol](https://www.analog.com/en/analog-dialogue/articles/uart-a-hardware-communication-protocol.html)
- read [Baud rate](https://solace.com/blog/what-is-baud-rate-bit-rate/)
- familiarize with [Orders_of_magnitude_(time)](https://en.wikipedia.org/wiki/Orders_of_magnitude_(time))
  - 10^(-6)) seconds
- The shortest interval is 32.02 microseconds so if we use this formula to find the Bit rate (bit/s) = 1 second / (32.02 x 10^(-6)) seconds = 31,230 bit/s
- familiarize how to change the bit rate in logic2 and change to Ascii.
